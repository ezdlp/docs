# NotNoise Documentation

Welcome to the NotNoise documentation! NotNoise is a smart music marketing platform that helps indie musicians promote their releases with professional landing pages, playlist campaigns, fan data collection, and more. This documentation covers all major features of NotNoise and guides you through using the web app step by step.
